title: Security
---

Xe looks to provide peace of mind with our clients when it comes to security and encryption, we regularly audit our selves and use the best-in-class security tools and practices to maintain a high level of security.

**HTTPS for secure connections**

Xe forces HTTPS for all services using TLS 1.2+ (SSL), including all public facing websites.

We continually audit our solutions using penetration testing, code analysis and the certificates we serve, the certificates authorities we use and we only use the strongest ciphers.
